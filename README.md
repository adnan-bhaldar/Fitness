### Run Locally

To run **Fitness** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/adnan-bhaldar/Fitness.git
```

Windows:

```bash
git clone https://github.com/adnan-bhaldar/Fitness.git
```
